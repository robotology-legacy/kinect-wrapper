This module runs the server part of the kinectWrapper library.

Description

A module that runs a kinectWrapperServer in order to take all the info from a kinect device.

Libraries

  • YARP libraries.
  • kinectWrapper library.

Parameters

–verbosity verbosity

  • specify the verbosity level of the server print-outs.

–period period

  • server thread period in [ms].

–name name

  • name of the server.

–img_width img_width

  • width of the rgb image to send.

–img_height img_height

  • height of the rgb image to send.

–seatedMode seatedMode

  • if put inside the options, the kinect device will be opened in seated mode (only if the driver is SDK).

–remap remap

  • if OpenNI, it provides depth aligned with rgb

–device device

  • kinect or xtion

Tested OS

Windows, Linux
