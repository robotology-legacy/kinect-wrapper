virtual bool kinectWrapper::KinectDriver::initialize ( yarp::os::Property &  opt)
pure virtual

Configure the driver.

Parameters
optcontains the set of options in form of a Property object.

Available options are:

info <string>: example (info KINECT_TAGS_ALL_INFO), specifies the info that is wanted to retrieve. If the parameter is equal to KINECT_TAGS_ALL_INFO, depth image, rgb image and skeleton information are retrieved. The alternatives are KINECT_TAGS_DEPTH, KINECT_TAGS_DEPTH_PLAYERS, KINECT_TAGS_DEPTH_RGB, KINECT_TAGS_DEPTH_RGB_PLAYERS and KINECT_TAGS_DEPTH_JOINTS.

seatedMode <bool>: if the property contains this value, seatedMode will be true, otherwise will be false. It can be true only if Microsoft SDK is used. If it is true, only upper body joints are tracked.

image_width <int>: example (image_width 320), specifies the width of the rgb image to send.

image_height <int>: example (image_height 240), specifies the height of the rgb image to send.

Returns
true/false if successful/failed.
